# Methodology

## M13 — Evidence Quality

Each finding is rated on a four-tier scale:

| Tier | Label | Meaning |
|---|---|---|
| (a) | Anthropic-official | Direct Anthropic documentation or announcement |
| (b) | Community-confirmed | Multiple independent sources converge on the same finding |
| (c) | Single-source anecdote | One developer report or forum post |
| (d) | Inference | Logical deduction from related facts, no direct evidence |

Findings rated (c) or (d) are flagged as requiring verification before the architecture spec is patched with normative language.

## Search Strategy

1. Anthropic official docs and blog
2. GitHub issues/discussions for claude.ai skill development
3. Reddit (r/ClaudeAI, r/anthropic)
4. Developer blogs and community reports
5. Publicly available SKILL.md examples

## Framework

RISEN (Role, Information, Steps, Expectation, Narrowing) — per prompt.md.
