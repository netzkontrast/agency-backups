# Research Root

**What is this folder?** The root directory for all deep-dive research tasks executed by agents in this repository.
**Why is it here?** To keep investigative, non-production code (like specs, literature reviews, and synthesis) organized and cleanly separated from the main codebase.

## Contents
- [ncp-novel-co-authoring-spec/](./ncp-novel-co-authoring-spec/): The task folder dedicated to researching and defining the Narrative Context Protocol and Dramatica integration for AI agents.
