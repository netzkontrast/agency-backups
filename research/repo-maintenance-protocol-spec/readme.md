# Research Task: Repository Maintenance Protocol and Dynamic Documentation Standard

This folder contains the research execution for defining the Nightly Maintenance Protocol, Dynamic Readmes, and the `/todo/` pipeline delegation.

## Subdirectories
- [workspace/](./workspace) - Scratchpad and session logs.
- [synthesis/](./synthesis) - Structured findings, state, and methodology.
- [reflection/](./reflection) - Cognitive trace logs and friction logs.
- [output/](./output) - Final specifications and FOLDERS.md update proposals.

## Assumptions
- We assume that updating `FOLDERS.md` logic is best done by generating a proposal (`proposed_FOLDERS_update.md`) rather than directly merging during this task to keep the scope of "research output" separate from "active state mutation" unless explicitly commanded (like `MAINTENANCE.md`).
