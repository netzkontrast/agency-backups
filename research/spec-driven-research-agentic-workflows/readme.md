# Spec-Driven Research Agentic Workflows

This directory contains the complete lifecycle artifacts for researching the meta-governance of agentic workflows. It adheres to the `RESEARCH.md` schema.

## Contents
- [`prompt.md`](./prompt.md) - The original explicit instructions.
- [`workspace/`](./workspace/) - Temporary logs and session history.
- [`synthesis/`](./synthesis/) - Synthesis logs and state tracking.
- [`reflection/`](./reflection/) - Frustration logs and meta-learning outputs.
- [`output/`](./output/) - The final compiled specification.

## Implicit Assumptions
- Assumed that creating python mock scripts for Google searches accurately simulates the required research process within a non-networked or simulated sandbox.
