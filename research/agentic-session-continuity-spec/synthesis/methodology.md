## Pre-Commitments (to be written by executing agent at S0)

The executing agent MUST record explicit pre-commitments for each Spec at S0 (Kickoff
Reflection) before beginning research. Pre-commitments are hypotheses that, if falsified,
would change the normative positions in the Specs.

Three hypothesis seeds are provided in the prompt (G-H0, H-H0, I-H0).
