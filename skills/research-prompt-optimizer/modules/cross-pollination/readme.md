# cross-pollination/

The 6 cross-pollination blocks — pairwise instructions for borrowing techniques across the three categories (A↔B, A↔C, B↔C).

## What and why

Phase 2 always selects exactly two cross-pollination blocks, paired and locked per category in `catalog.yaml` → `cross_pollination_pair`. The user can swap which two via the edit branch but cannot remove one.
